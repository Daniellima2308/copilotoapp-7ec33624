# Sentinela

Pasta reservada para os assets oficiais da **marca principal** do app.

## Estrutura oficial
- `logo/`
- `icone/`
- `simbolo/`
- `wordmark/`
- `slogan/`
- `reference/`

## Convenção oficial de nomenclatura
Todos os arquivos da marca Sentinela devem seguir:

`sentinela-[tipo]-[variacao]-[versao].png`

> Mudança oficial: a convenção passou de `[cor]` para `[versao]`, pois a identidade principal é multicolorida por natureza.

### Regras
- usar tudo em minúsculo
- usar hífen (`-`)
- sem espaços
- sem acentos
- sem nomes vagos (`final`, `final2`, `novo`, `teste`)
- não usar `dark/light/primary/alt` como convenção principal

## Tipos oficiais (`[tipo]`)
- `logo`
- `icone`
- `simbolo`
- `wordmark`
- `slogan`
- `reference`

## Variações oficiais (`[variacao]`)
- `oficial`
- `app`
- `com-slogan`

## Versões oficiais (`[versao]`)
- `institucional` = versão oficial multicolorida da marca
- `branca` = versão para fundos escuros
- `preta` = versão monocromática escura para fundos claros

## Composição da marca
- `icone` = símbolo dentro do quadrado arredondado escuro, usado como ícone do app
- `simbolo` = símbolo sozinho, sem wordmark
- `logo` = símbolo + wordmark
- `logo com slogan` = símbolo + wordmark + slogan
- `wordmark` = apenas `SENTINELA`
- `slogan` = apenas a frase institucional
- `reference` = material de referência visual e legado
