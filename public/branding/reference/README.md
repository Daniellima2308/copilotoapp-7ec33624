# Branding reference

Material oficial de referência visual geral do projeto para consulta.

## Escopo
- Pode conter guias visuais e referências institucionais.
- Não substitui os assets oficiais de `sentinela/` (marca principal) ou `bino/` (personagem/assistente).

## Diretriz de prioridade
- A marca principal do app é a Sentinela.
- Bino permanece como personagem/assistente separado.
- A pasta `reference/` é apenas apoio visual e histórico, não fonte primária de asset oficial de marca.

## Observação
Quando houver asset oficial de marca, priorize salvar no diretório específico da marca correspondente.
