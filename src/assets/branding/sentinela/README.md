# Branding oficial SENTINELA

Esta pasta foi preparada para receber os assets oficiais finais da marca SENTINELA e do personagem Bino.

## Pendências de asset final
- wordmark isolado escuro para fundo claro
- wordmark isolado claro para fundo escuro
- ícone oficial
- ícone oficial versão 2
- logo oficial somente wordmark para fundo claro e sem slogan
- logo oficial com wordmark branco e slogan para fundo escuro
- Bino oficial 1
- Bino oficial 2
- Bino oficial 3
- referência oficial

## Observação
Enquanto os arquivos finais não forem disponibilizados dentro do repositório, a aplicação usa fallbacks vetoriais temporários em `public/branding/sentinela` para remover a marca antiga sem reinventar o produto visual completo.
