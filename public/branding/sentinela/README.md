# Sentinela

Pasta reservada para os assets oficiais da **marca principal** do app.

## Estrutura oficial
- `logo/`
- `icone/`
- `simbolo/`
- `wordmark/`
- `slogan/`
- `reference/`

## Convenção oficial de nomenclatura
Todos os arquivos da marca Sentinela devem seguir:

`sentinela-[tipo]-[variacao]-[cor].png`

### Regras
- usar tudo em minúsculo
- usar hífen (`-`)
- sem espaços
- sem acentos
- sem nomes vagos (`final`, `final2`, `novo`, `teste`)
- não usar `dark/light/primary/alt` como convenção principal

## Tipos de asset (`[tipo]`)
- `logo`
- `icone`
- `simbolo`
- `wordmark`
- `slogan`
- `reference`

## Variações de composição (`[variacao]`)
- `oficial`
- `com-icones`
- `sem-icones`
- `com-slogan`
- `sem-slogan`

> Observação: a variação pode combinar mais de um termo, por exemplo `oficial-com-icones`.

## Cores oficiais (`[cor]`)
- `branco`
- `preto`
- `azul-petroleo`
- `azul-claro`

## Exemplos válidos
- `sentinela-logo-oficial-com-icones-azul-petroleo.png`
- `sentinela-logo-oficial-com-icones-preto.png`
- `sentinela-logo-oficial-com-icones-branco.png`
- `sentinela-logo-oficial-com-icones-azul-claro.png`
- `sentinela-simbolo-oficial-sem-icones-azul-petroleo.png`
- `sentinela-wordmark-oficial-branco.png`
- `sentinela-slogan-oficial-preto.png`

## Importante
- Esta pasta é somente da marca Sentinela.
- Não incluir assets do personagem Bino neste diretório.
