# Sentinela

Pasta reservada para os assets oficiais da **marca principal** do app.

## Estrutura oficial
- `logo/`
- `icone/`
- `simbolo/`
- `wordmark/`
- `slogan/`
- `reference/`

## Convenção oficial de nomenclatura (regra híbrida)
A nomenclatura oficial da Sentinela é híbrida por tipo de asset:

- `logo`, `icone`, `simbolo` e `reference` usam:
  - `sentinela-[tipo]-[variacao]-[versao].png`
- `wordmark` e `slogan` usam:
  - `sentinela-[tipo]-[variacao]-[cor].png`

> Diretriz oficial: composição multicolorida por natureza em `logo/icone/simbolo/reference`, mantendo `wordmark/slogan` orientados por cor.

### Regras
- usar tudo em minúsculo
- usar hífen (`-`)
- sem espaços
- sem acentos
- sem nomes vagos (`final`, `final2`, `novo`, `teste`)
- não usar `dark/light/primary/alt` como convenção principal

## Tipos oficiais (`[tipo]`)
- `logo`
- `icone`
- `simbolo`
- `wordmark`
- `slogan`
- `reference`

## Variações oficiais (`[variacao]`)
- `oficial`
- `app`
- `com-slogan`

## Versões oficiais (`[versao]`)
> Aplicável a: `logo`, `icone`, `simbolo`, `reference`

- `institucional` = versão oficial multicolorida da marca
- `branca` = versão para fundos escuros
- `preta` = versão monocromática escura para fundos claros
- `oficial` = versão única quando o tipo não exigir desdobramento visual adicional

## Cores oficiais (`[cor]`)
> Aplicável a: `wordmark`, `slogan`

- `azul-petroleo`
- `branco`
- `preto`

## Composição da marca
- `icone` = símbolo dentro do quadrado arredondado escuro, usado como ícone do app
- `simbolo` = símbolo sozinho, sem wordmark
- `logo` = símbolo + wordmark
- `logo com slogan` = símbolo + wordmark + slogan
- `wordmark` = apenas `SENTINELA`
- `slogan` = apenas a frase institucional
- `reference` = material de referência visual e legado
