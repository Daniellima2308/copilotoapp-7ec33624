# Branding assets

Estrutura oficial dos assets de branding do projeto.

## Domínios de branding
- `sentinela/` = marca principal do app (identidade institucional).
- `bino/` = personagem/assistente do app (não substitui a marca principal).
- `reference/` = material oficial de referência visual geral para consulta.

## Regras de separação
- Não misturar assets da Sentinela com assets do Bino.
- A convenção de nomes da marca principal fica definida em `sentinela/README.md`.
- Materiais de referência não devem sobrescrever os arquivos oficiais de marca.
